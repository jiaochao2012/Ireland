# Ireland Tourism Website

一个展示爱尔兰旅游信息的现代化网站，提供景点介绍、旅游攻略、酒店预订等功能。

## 功能特点

- 🌍 景点展示：展示爱尔兰著名景点信息
- 📝 旅游攻略：提供详细的旅游攻略和行程建议
- 🏨 酒店预订：查看和预订酒店
- 🍽️ 美食推荐：推荐当地特色美食和餐厅
- 🌐 多语言支持：支持中文和英文
- 📱 响应式设计：完美适配各种设备

## 技术栈

- Vue.js 3
- Vue Router
- Vuex
- SCSS
- ESLint
- Babel

## 项目设置

### 安装依赖
```bash
npm install
```

### 开发环境运行
```bash
npm run serve
```

### 生产环境构建
```bash
npm run build
```

### 代码检查
```bash
npm run lint
```

## 项目结构
